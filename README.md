# 🎭 Deepfake Detection System using ResNeXt & LSTM

This project focuses on detecting deepfake videos using deep learning techniques.

---

## 🚀 Overview

* Detects deepfake videos using AI
* Uses transfer learning with pretrained ResNeXt
* Extracts frame-level features from videos
* Processes temporal data using LSTM
* Integrated with a Django web application

---

## 🧠 Methodology

1. Video is split into frames
2. Frames are passed through ResNeXt CNN
3. Feature vectors are extracted
4. Features are fed into LSTM network
5. Model predicts whether video is Real or Fake
